# ✅ To-Do List App

A minimal and clean To-Do List web app built using **HTML**, **CSS**, and **JavaScript**, with support for local storage so your tasks stay even after you refresh the page.

## 🔗 Live Demo

👉 [View it Live](https://praveen-to-do-tracker.netlify.app/)

*Replace the above link with your actual Netlify deployment URL.*

## 🎯 Features

- Add and display tasks
- Click on a task to mark it as completed (toggle)
- Right-click on a task to delete it
- Stores tasks in **localStorage**
- Simple and responsive UI
- Mobile-friendly layout



## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API
